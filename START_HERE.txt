🚀 QUICK START (5 MENIT):

LANGKAH 1: Setup Database
  $ mysql -u root -p < db_koperasi.sql

LANGKAH 2: Update Password (jika ada)
  Edit: src/Koperasi/Koneksi.java
  Ubah: String pass = "password_anda";

LANGKAH 3: Jalankan Aplikasi
  - NetBeans: Klik kanan project → Run
  - Command:  javac & java (lihat QUICK_REFERENCE.md)

LANGKAH 4: Test
  - Klik Tambah, Update, Delete, Clear
  - Semua harus berfungsi dengan baik ✅
